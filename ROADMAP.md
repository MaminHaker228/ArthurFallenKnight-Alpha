# Arthur: Fallen Knight - Development Roadmap

## Version 0.1 ALPHA (Current) ✅

**Status:** Core gameplay implemented, placeholder assets

### Completed Features
- [x] Player controller (movement, dash)
- [x] Combat system (attack combos, block, parry)
- [x] Enemy AI (patrol, chase, melee/ranged attacks)
- [x] Mini-boss implementation (2-phase Twisted Knight)
- [x] Dialogue system (JSON-driven, branching ready)
- [x] Inventory system (items, equipment, rarity tiers)
- [x] Localization (Russian & English)
- [x] Particle effects framework
- [x] Camera shake system
- [x] One playable level (Cursed Forest)
- [x] Complete documentation (GDD, README, code comments)
- [x] CI/CD pipeline (GitHub Actions auto-builds)

### Known Limitations
- ⚠ Placeholder graphics (needs 32x32 pixel sprites)
- ⚠ No audio (music & SFX system ready, needs content)
- ⚠ Single level
- ⚠ Basic animations only
- ⚠ Limited enemy types (3 base + boss)
